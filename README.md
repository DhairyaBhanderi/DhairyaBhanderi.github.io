# Crafted Narrative

An immersive single-page experience built with React, Vite, and Tailwind CSS. The site highlights projects, credentials, and animated storytelling components.

## Getting started

### Prerequisites
- Node.js (LTS recommended) and npm

### Installation
```sh
git clone <REPOSITORY_URL>
cd <REPOSITORY_DIRECTORY>
npm install
```

### Development
Start a local dev server with hot reloading:
```sh
npm run dev
```

### Production build
Create an optimized build:
```sh
npm run build
```
Preview the production build locally:
```sh
npm run preview
```

## Tech stack
- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Framer Motion & Three.js for interactive elements

## Project structure
- `src/components`: UI and feature components
- `public`: Static assets served at the root path
- `vite.config.ts`: Vite configuration and aliases
